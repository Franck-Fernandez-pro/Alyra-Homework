## Hardhat

- Hardhat
- Goerli

Run blockchain:

```bash
$ cd hardhat/
$ npm i
$ npx hardhat node
```

Deploy on goerli:

```bash
$ cd hardhat/
$ npm i
$ npx hardhat run scripts/deploy.ts --network goerli
```
